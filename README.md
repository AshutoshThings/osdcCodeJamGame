# Down to Earth Santa

A winter-themed delivery game built with JavaScript and HTML5 Canvas. Play as Santa on a sleigh, delivering packages to houses while avoiding falling ice and the Grinch.

## Gameplay

- Collect packages from the warehouse
- Deliver to marked houses before time runs out
- Avoid falling ice blocks (damages health)
- Jump on the Grinch to defeat him (requires packages)
- Use turbo power-ups for speed boosts
- Collect health power-ups to restore hearts

## Controls

| Key | Action |
|-----|--------|
| A/D or Arrow Keys | Move left/right |
| W/Space | Jump |
| Shift | Sprint |
| P | Pause |
| ESC | Return to menu |

## Features

- Procedurally generated levels
- AI-powered level generation (Easy/Medium/Hard)
- Dynamic weather with parallax backgrounds
- Health and stamina system
- Combo scoring system
- Minimap navigation

## Running the Game

1. Start the level server:
   ```
   node level-server.js
   ```

2. Open `winter-courier.html` in a browser

## Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- Node.js (level server)
- Groq API (AI level generation)
